# Lab 0B: Model Warm-Up and Comparison

## Purpose

Use this warm-up lab after [lab0_1_environment_setup/03_environment_check.ipynb](../lab0_1_environment_setup/03_environment_check.ipynb) is working. The goal is to give you one small hands-on AI task before you start the four forensic pattern labs.

## Learning Goals

By the end of this warm-up lab, you should be able to:

- discover available models from the configured Ollama endpoint
- run the same prompt on 3 different models
- compare model outputs using a simple scoring rubric
- notice differences in structure, accuracy, and response time

## What To Do

Complete the steps in this order:

1. Finish [lab0_1_environment_setup/03_environment_check.ipynb](../lab0_1_environment_setup/03_environment_check.ipynb).
2. Open [02_model_comparison.ipynb](02_model_comparison.ipynb).
3. Run the notebook from top to bottom.
4. Use the default model from `.env` and pick two additional models from the same Ollama endpoint.
5. Compare the three models on the synthetic PII extraction task.
6. Complete the short reflection at the end of the notebook.

## Success Criteria

You have completed this warm-up lab when:

- you run the notebook successfully
- you compare 3 different models on the same task
- you review the score summary
- you complete the reflection section
